# 全球財經趨勢儀表板 (Global Market Dashboard)

這是一個基於 Python 與 Streamlit 框架開發的互動式金融數據監控平台。本應用程式整合了 Yahoo Finance 與台灣銀行 (Bank of Taiwan) 的數據來源，提供使用者一個簡潔、即時且可互動的介面，用以追蹤全球主要股票指數、各國匯率及關鍵原物料的價格走勢。

## 專案特色

### 1. 全球市場總覽 (Market Overview)
- 採用網格化儀表板設計，單一頁面即時呈現六大關鍵指數：
  - 台灣加權指數 (TWSE)
  - 美股四大指數 (S&P 500, Nasdaq, Dow Jones, SOX)
  - 日經 225 指數 (Nikkei 225)
- 每張圖表皆包含即時價格、漲跌幅數據以及歷史走勢小圖。

### 2. 多元資產追蹤
- **各國匯率**：直接爬取台灣銀行牌告匯率 (現金/即期)，確保數據貼近台灣使用者需求。
- **國際原物料**：串接 Yahoo Finance API，提供黃金、白銀、原油、天然氣及農產品(黃豆、玉米、小麥)等期貨價格。
- **比較分析**：支援多重項目選取，可同時疊加比較不同資產的走勢。

### 3. 進階互動圖表
- 使用 Plotly 繪圖引擎，提供滑鼠懸停資訊顯示。
- **內建繪圖工具**：圖表整合了畫筆、直線、矩形標註與橡皮擦功能，使用者可直接在網頁上進行簡易的技術分析標註。
- **彈性時間軸**：透過側邊欄滑桿，可自由調整 7 天至 365 天的歷史數據觀察區間。

### 4. 自動化更新機制
- 內建 JavaScript 自動刷新腳本，網頁每 5 分鐘 (300秒) 自動重整。
- 搭配 Streamlit 快取 (Caching) 機制，優化數據載入效能，同時確保資訊即時性。
- 頁面右上角即時顯示最後更新時間。

## 技術架構

- **程式語言**：Python 3.x
- **核心框架**：Streamlit
- **數據處理**：Pandas, Beautiful Soup 4 (網頁爬蟲)
- **數據來源**：yfinance (Yahoo Finance API), Requests (台灣銀行網頁)
- **視覺化**：Plotly Express

## 安裝與執行說明

### 1. 環境準備
確認您的電腦已安裝 Python。建議建立一個虛擬環境來執行本專案。

### 2. 安裝必要套件
請將以下套件清單儲存為 requirements.txt，並執行安裝指令：

```txt
streamlit
pandas
requests
yfinance
plotly
beautifulsoup4
```

接著在終端機執行安裝指令：

```python
pip install -r requirements.txt
```
### 3. 啟動應用程式
在終端機 (Terminal) 中執行以下指令：
```python
streamlit run app.py
```
啟動後，瀏覽器將自動開啟並顯示儀表板
